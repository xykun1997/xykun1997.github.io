shell入门——基本运算符

shell基本运算符

类别介绍

  • 算术运算符
  • 关系运算符
  • 布尔运算符
  • 字符串运算符
  • 文件测试运算符

几点注意

  • 在进行求值操作时要使用expr
  • 表达式和运算符之间必须要有空格
  • 算数表达式要完全被``包含

算数运算符

运算符 说明 使用举例
+ 加法 expr $a + $b
- 减法 expr $a - $b
* 乘法 expr $a \* $b
/ 除法 expr $a / $b
% 取余 expr $a % $b
= 赋值 a = $b 将变量b的值赋给a
== 判相等，相同为true [ $a == $b ]
!= 判不等，不等为true [ $a != $b ]

注意：

  1. 乘号(*)前边必须加反斜杠(\)转义才能实现乘法运算；
  2. 条件表达式要放在方括号之间，并且要有空格，例如: [$a==$b] 是错误的，必须写成 [ $a == $b ]。
    实例： 输出结果

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字
假设a=20; b=30.

运算符 说明 使用举例
-eq 两个变量相等 返回 true [ $a -eq $b ] 返回false
-ne 两个变量不相等 返回 true [ $a -ne $b ] 返回true
-gt 左变量 > 右变量 返回true [ $a -gt $b ] 返回false
-lt 左变量 < 右变量 返回true [ $a -lt $b ] 返回true
-ge 左变量 >= 右变量 返回true [ $a -ge $b ] 返回false
-le 左变量 <= 右变量 返回true [ $a -le $b ] 返回true
注意
表达式要放在方括号之间，并且要有空格，[ $a -gt $b ]。

字符串运算符

假设a=“zyk”;b=“xy”.

运算符 说明 使用举例
= 两个字符串相等 返回true [ $a = $b ] 返回false
!= 两个变量不相等 返回 true [ $a ！= $b ] 返回true
z 字符串长度为0 返回true [ -z $a ] 返回false
n 字符串长度不为0 返回true [ -n $a ] 返回true
str 字符串不为空 返回true [ $a ] 返回true

文件测试运算符

file=”/shell-script/sh02.sh”

运算符 说明 使用举例
-b 文件是块设备文件 返回true [ -b $file ]返回false
-c 文件是字符设备文件 返回 true [ -c $file ]返回false
-d 文件是目录文件 返回true [ -d $file ]返回false
-f 文件是普通文件(不是目录也不是设备文件)返回 true [ -f $file ]返回true
-g 文件设置了SGID位 返回true [ -g $file ]返回false
-k 文件设置了粘着位 返回true [ -k $file]返回false
-p 文件是有名管道 返回true [ -p $file ]返回false
-u 文件设置了SUID位 返回true [ -u $file ]返回false
-r 文件可读 返回true [ -r $file ]返回 true
-w 文件可写 返回true [ -w $file ] 返回true
-x 文件可执行 返回true [ -x $file ] 返回true
-s 文件不为空 返回true [ -s $file ] 返回true
-e 文件存在 返回true [ -e $file ] 返回true
